# Sistema Financiero ANF115

## Descripción
Este proyecto es un sistema financiero desarrollado para la gestión de operaciones financieras básicas. Está diseñado para ser ejecutado en un entorno local utilizando XAMPP.

## Requisitos
- XAMPP (Apache, MySQL, PHP)
- Navegador web moderno

## Instalación y Configuración
1. Clona o descarga este repositorio en la carpeta `htdocs` de tu instalación de XAMPP.
2. Inicia XAMPP y activa los módulos Apache y MySQL.
3. Accede al proyecto desde tu navegador en `http://localhost/sistema-financiero-anf115`.

## Uso
- Abre el proyecto en tu navegador.
- Navega por las diferentes secciones del sistema financiero.
